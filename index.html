About Me

I am a freelance web developer with over 7 years experience of building Javascript web-applications for large corporate and enterprise clients.

Having started out life coding on the LAMP stack, I have, since early 2008, specialised in Javascript development, becoming fluent in a number of frameworks and libraries on both the client and the server as well as developing clients' bespoke DOM and UI libraries written in native Javascript.

In recent years I have been focussed on building robust, resilient and scalable software applications based on NodeJS. This has involved making a number of contributions back into the projects used, and also developing and releasing my own open-source tools through npm.

Over the course of my career, I have worked for a number of large, high-profile end-clients, including Barclays, British Gas, BSkyB, GOV.UK, The Home Office, HSBC, JWT, Lloyds TSB, Marks & Spencer, O2, RBS Group, Santander, Unilever and Virgin Atlantic.

Open Source Contributions

Independently authored of a number of open source npm modules including:

  • i18n-future - Translation and multi-language content for Node. Originally inspired by the i18next project, with a module-safe port of the API to use in a Node environment. Used in production by a number of projects in The Home Office.
  • npm-sass - A wrapper for node-sass which has 'npm install path'-aware `includePaths` settings, so that npm modules can be safely used in sass projects with complex dependency trees. Used in production by a number of projects in The Home Office.
  • reqres - Mock express request and response objects for testing express routers and middlewares.
  • nimm - A CLI wrapper around a node process to detect when the process fails due to missing dependencies and prompts to install.
  • moment-business-time - A plugin for the moment date library for performing calculations with business days and hours.
  • express-healthcheck - Simple express middleware for creating healthcheck endpoints.
  • busboy-body-parser - Middleware for express wrapping the busboy multipart form handler to parse multipart form submissions in a similar way to express default body parsers.
